# ☢️ CourseChooseBoomProPlusMax++

西电选课系统抢课助手。脚本运行在选课页面内，复用页面已有登录态和请求环境，自动保存待选课程，并按配置轮询提交选课请求。

## 功能

- 添加课程号和班级号，班级默认 `01`，支持 `01` 到 `05`。
- 添加课程时自动解析并保存 `clazzId`，解析失败不会写入本地列表。
- 打开选课页面后会刷新本地课程的 `clazzId` 和 `secretVal`。
- 抢课时只使用已缓存的 `clazzId` 发包，成功状态以已选课程接口确认为准。
- 支持自定义轮询间隔。
- 支持跳过当前 pending 请求并进入下一轮。
- 普通模式下并发固定为 `1`，避免过高请求量影响稳定性。
- 内置一个可提升性能的神秘彩蛋。
- 面板宽高可拖拽调整，配置会自动保存。

## 安装

1. 安装 Tampermonkey、Violentmonkey 或其他 userscript 管理器。
2. 新建脚本，粘贴 `main.js` 内容并保存。
3. 打开西电选课页面：

```text
https://xk.xidian.edu.cn/xsxk/elective/grablessons*
```

脚本会在页面就绪后自动显示控制面板。

## 使用

1. 输入课程号。
2. 选择班级号。
3. 点击添加。
4. 设置轮询间隔。
5. 点击开始抢课。

脚本会循环发起选课请求，并通过已选课程接口确认是否真正成功。只有确认成功的课程才会被标记为完成。

## 数据兼容

脚本使用浏览器 `localStorage` 保存配置。旧版本保存的课程列表会继续兼容；没有班级号的旧课程会按 `01` 处理，并在页面启动时重新刷新课程对应的 `clazzId`。
